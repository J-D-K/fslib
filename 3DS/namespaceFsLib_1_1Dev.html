bool FsLib::Dev::InitializeSDMC ( void  )

Initializes a bare-bones compatibility layer so devkitPro libraries still work with the SD card of the 3DS.

Returns
True on success. False on failure.
Note
In order to use this correctly, libctru's __appInit(void) and __appExit(void) must be overriden to not include archiveMountSdmc() and archiveUnmountAll. This will only function for the sdmc and only for reading and writing files. This is only in place so devkitPro libraries can still function and access files on sdmc. All other operations should use the functions FsLib provides, FsLib::File, and FsLib::Directory.
